FinGuard is a Python-based fraud analytics application designed to simulate how financial institutions identify and investigate potentially fraudulent transactions. The project processes 10,000+ simulated banking transactions, assigns risk scores using rule-based fraud detection logic, generates professional Excel investigation reports, and creates analytical dashboards and visualizations for decision-making.

Features
Processes 10,000+ simulated financial transactions
Rule-based fraud risk scoring engine
Automated fraud investigation reports
Executive Excel dashboards
Data visualization using Matplotlib
Risk classification (Low, Medium, High)
Automated recommended analyst actions
Transaction trend analysis
Professional Excel formatting with OpenPyXL
