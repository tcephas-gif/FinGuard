# FinGuard Fraud Analytics System

A Python-based fraud analytics system that simulates the workflow of a financial fraud analyst by processing over 10,000 simulated banking transactions, assigning fraud risk scores, generating investigation recommendations, creating executive dashboards, and exporting professional Excel reports.

---

# Features

- Processes and analyzes 10,000+ simulated financial transactions
- Rule-based fraud detection engine with weighted risk scoring
- Automated investigation recommendations
- Executive KPI dashboard generated in Excel
- Professional conditional formatting for high-risk transactions
- Interactive analytics visualizations using Matplotlib
- Automated report generation with OpenPyXL
- Business intelligence reporting for fraud trends

# How It Works

1. Generate a realistic financial transaction dataset.
2. Apply fraud detection rules using weighted risk scoring.
3. Classify transactions as Low, Medium, or High Risk.
4. Generate investigation reasons and recommended analyst actions.
5. Build executive dashboards and KPI summaries.
6. Export a professional Excel investigation report.
7. Save visual analytics charts automatically.
